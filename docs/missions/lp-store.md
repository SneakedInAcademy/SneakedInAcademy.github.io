# Loyalty Points (LP) und LP Store

Viele Missionen in EVE Online geben neben ISK auch **Loyalty Points (LP)**.

Diese Punkte können im **LP Store einer NPC Corporation** gegen besondere Items eingetauscht werden.

---

# Was sind Loyalty Points?

Loyalty Points sind eine Art **Belohnungssystem für Mission Runner**.

Sie erhältst du durch:

- Missionen
- Storyline Missionen
- bestimmte Events

LP sind immer an **eine bestimmte NPC Corporation gebunden**.

Beispiel:

LP von **Caldari Navy** können nur im **Caldari Navy LP Store** genutzt werden.

---

# Was man im LP Store kaufen kann

Im LP Store können viele verschiedene Items gekauft werden.

Typische Beispiele:

- Module
- Implantate
- Blueprint Copies
- Faction Ammunition
- Faction Ships

Viele dieser Items sind **wertvoller als die normale Missionsbelohnung**.

---

# Wie der LP Store funktioniert

Items im LP Store kosten meist eine Kombination aus:

- Loyalty Points
- ISK
- bestimmten Items oder Tags

Beispiel:

| Item | Kosten |
|---|---|
| Faction Module | LP + ISK |
| Implantat | LP + ISK |
| Faction Ship | LP + ISK + Tags |

---

# LP in ISK umwandeln

Viele Spieler versuchen, ihre LP möglichst **effizient in ISK umzuwandeln**.

Das geschieht durch:

1. Item im LP Store kaufen
2. Item im Markt verkaufen

Der Wert von LP wird oft als **ISK pro LP** berechnet.

Typische Werte:

| Qualität | ISK pro LP |
|---|---|
| schlecht | unter 500 |
| durchschnittlich | 800–1200 |
| sehr gut | 1500+ |

Der tatsächliche Wert hängt stark vom **Marktpreis der Items** ab.

---

# Tipps für neue Spieler

- sammle zunächst **LP bei einer einzigen NPC Corporation**
- prüfe den Marktwert eines Items, bevor du es kaufst
- manche LP Store Items bringen deutlich mehr ISK als andere

Der LP Store ist für viele Mission Runner eine der **wichtigsten Einnahmequellen**.
